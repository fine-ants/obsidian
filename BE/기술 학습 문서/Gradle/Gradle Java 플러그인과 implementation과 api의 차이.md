

## 1. Gradle Java 플러그인과 Java-Library 플러그인
### Java 플러그인은 무엇인가?
Gradle 설정에서 Java 플러그인을 추가하면 해당 프로젝트를 자바 프로젝트로 만들고 자바 소스코드를 컴파일, 테스트, 빌드하는데 필용한 기능들을 Gradle에 부여해줍니다.

```gradle
plugins {  
    id 'java'
}
```
