Authorization Code Grant Type은 OAuth2.0 Grant Type들 중에서 가장 일반적인 유형입니다. 해당 타입은 웹 애플리케이션과 네이티브 앱 모두에서 사용자가 애플리케이션을 승인(Grant)한 후 액세스 토큰을 얻는데 사용됩니다.
