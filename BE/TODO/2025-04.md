문제
- 로컬 서버 실행시 외부 API를 통해서 환율을 가져옵니다. 그런데 API 요청이 고갈되면 별도의 대응이 없다. 결과적으로 베이스 환율을 제외한 다른 환율들은 0으로 설정됩니다.
해결방법
- 로컬 서버로 실행시에는 파일 기반으로 작동하던가? 아니면 mock api를 구성해서 랜덤하게 반환하는 방법이 있습니다.
